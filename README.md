#The C++ Basics Exam
Данный репозиторий содержит учебные проекты, созданные для повторения основ языка C++. В ходе работы были закреплены следующие темы: работа с массивами и строками в стиле C, ввод и вывод с использованием cin и cout, функции, условные операторы, циклы, а также алгоритмы сортировки (пузырьком), поиска и обработки пользовательского ввода.

Менеджер задач
Программа представляет собой консольное приложение для управления списком задач. Все данные хранятся в статических массивах фиксированного размера. Каждая задача содержит название, описание, приоритет, дату и время.

Функционал приложения:

Добавление новой задачи с вводом всех параметров

Удаление задачи по её индексу в списке

Редактирование существующей задачи

Поиск задач по названию, приоритету, описанию, дате или времени

Сортировка задач по приоритету (по убыванию) или по дате (по возрастанию)

Просмотр задач за определённый период (день, неделя, месяц) с возможностью выбора сортировки

Калькулятор выражений
Программа выполняет вычисление арифметических выражений, введенных пользователем в виде строки. Поддерживаются целые числа, операции сложения, вычитания, умножения и деления, а также круглые скобки и унарный минус.

Принцип работы:

Разбор входной строки с выделением чисел и операторов

Использование двух стеков: для чисел и для операторов

Учёт приоритета операций при вычислении

Обработка скобок и унарного минуса

Последовательное выполнение операций до получения конечного результата
