# The-C-Basics-Exam
Повторили основы C++: работу с массивами и строками (в стиле C), ввод‑вывод (cin/cout), функции, условные операторы и циклы. Закрепили сортировку (пузырьком), поиск, обработку ввода. В менеджере задач — добавление, удаление, редактирование данных; в калькуляторе — парсинг строк, стеки, приоритеты операторов и обработку скобок.
